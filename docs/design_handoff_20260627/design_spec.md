# 设计规格 ·《中国非遗走出去了。但，被看到了吗？》

给 Claude Design 继续微调用。这份是"为什么这么设计 + 还要调什么"，token 的硬数值见 `design_tokens.css`。

## 一、概念

**主题**：传统 × 算法的碰撞。中国非遗（文化重量：宋体、窗花、朱砂）撞进 TikTok 的算法之海
（电光青、等宽数字、噪点、RGB 故障）。设计语言本身就是论点的一部分——身份在信号里被损坏。

**口吻**：大胆、不无聊、不怕丑。退掉"非遗=博物馆"的暖棕温吞安全套路。大白话，退黑话
（"触达区间"→"真正被认出的有多少"；"撞词率"→"搜到的里有多少根本不是它"）。

## 二、衡量口径（红线·不可跑偏，详见 方案_v4.md 第 1 节）

- 我们测的是**海外能见度/曝光**，**不是影响力**。措辞用"被看见的程度"。
- **2D 口径（终态 2026-06-27）：绝对真实播放 = 标尺（能见度本体，378核查锚定）；份额 = 透镜（是否被淹），绝不给隐形排名。**
  反例铁律：书法份额仅 12% 看着"被淹"，但绝对 **8500 万**真实播放 = 被大量看见。份额会骗人，绝对量不会。
- **四象限**（绝对×份额，44项落格）：①独占且被看见15 ②挤着但被看见4(书法/呼麦/宣纸/珠算) ③独占但小众18(藏戏/藏医/福船) ④真隐形7(剪纸/云锦/格萨尔)。
  数据在 `figures_data.json` 每项 `reach_anchored`/`share_real_pct`/`fate` + `fate_counts`。
- **两个透镜各司其职，都不预设任何词本该属于中国**：
  - **名字/锚点**（解释你在哪一行）：罗马音救不了(shufa 3%/fuchuan 3%)，通用译词被淹，**真正拉起份额的是地域/民族/节庆锚点的复合名**（celadon 52%→龙泉 73%）。锚点是结构属性不是营销杠杆。
  - **内容竞争力**（解释你在大赛道里能否从低份额捞到高绝对）：书法 8500万 vs 剪纸 1,414。
- ⛔ **删除"身份主张/identity claim"措辞**：papercut/calligraphy 本就各国共有，低份额 **≠** 身份失败。
- 识别实验**落点**：「份额会骗人，绝对量才是能见度：剪纸真隐形(1,414)、书法挤着但被看见(8500万)」→ 交棒塌缩+四象限；**绝不**落在"你认错了"或"它影响力低"。
- 福船：junk+watertight 是**我们查询用错的词**（官方名 "Chinese junks"），已从图表**剔除并认账**；福船真相是**缺席型**(~5万)，不是被淹没，**不当塌缩典型**。
- 诚实边界化进开篇+结尾各一句，**不设单独"方法与边界"章节**。

## 四、字体

- 中文展示/标题：**宋体**（文化重量）。
- 数字/英文/标签：**等宽 mono**（算法冷感，全站数字都用 mono）。
- 正文：sans。

## 五、招牌动作（让它"不无聊"的关键，别丢）

1. **RGB 错位故障**：标题关键词（如"被看到了吗"）红/青错位描边。
2. **巨号轮廓数字**：章节序号 01/02… 超大、镂空红描边。
3. **窗花 RGB 撕裂出血**：`assets/motif_papercut.svg`，放大、出血到边缘，做主视觉。
4. **硬投影偏移**：按钮/可点元素 hover 位移 + 实色投影。
5. **扫描线噪点**：全局叠加颗粒。
6. **破规整**：feed 墙卡片上下交错，不要对齐成死板网格。

## 六、两个招牌交互组件

- **识别实验**（开篇）：TikTok 文字封面卡（双调底 + 文案即视觉 + ▶播放量，**不用 emoji 占位**）；
  卡按真实播放排序，读者找"中国版排第几" → 揭晓 → 非中国卡盖红章「撞词」+ 灰掉，中国卡盖青章「真」+ 发光；
  banner 顶出反差（剪纸：真的那条 1,414 播放排 141 位 / 头顶 13.9M）。
  **两轮对比**：papercut（真隐形，绝对 1,414/份额 0%）vs calligraphy（**挤着但被看见，绝对 8500 万**/份额 12%/第3）——
  落点"**份额会骗人，绝对量才是能见度**"，不是"你认错了"。abacus 备选第三轮（核查回补后 9900万/26%）。数据：`recognition_rounds.json`（`cn_play_abs` 优先于 `cn_play_share_pct`）。
- **非遗身份证档案馆**（结尾）：44 张卡，**按四象限/绝对真实播放排**，点卡翻面看
  「名字 + 真实触达(绝对) + 象限诊断 + 一句话故事」。数据：`figures_data.json` 的 `projects`（`fate`/`reach_anchored`/`share_real_pct`）。

## 八、待微调（已知，交给 Claude Design 拍）

- **标题孤字**：「中国非遗走出去了。」的"了。"会掉到第二行——需要断行控制（手动 `<br>` 或调字号），别让它像没排好。
- 是否再野：卡片轻微倾斜、hover 抖动故障、噪点更重、字号更冲。样板是"够大胆"的起点，可继续推。
- 卡封面亮底（g3 青 / g4 酸黄）上的白字对比，需复核可读性（已加暗角 scrim）。
- 移动端：巨标题/feed 墙/巨号数字的断点（样板有基础 @media，需细化）。
- 字体：若 Design 能用 Web 字体，建议给宋体换更有个性的衬线/展示体，mono 换更现代的等宽（如 Space Mono 类），会更脱"系统默认感"。
