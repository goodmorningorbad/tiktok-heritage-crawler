# 人工审核结果按新口径复核：冲突项

共发现 5 条需要改标/复核的冲突项。

- **NE-15 福建木偶戏**：人工=relevant → 建议=irrelevant。理由：中国皮影戏是独立 UNESCO 项目，不能归入福建木偶戏。 URL: https://www.tiktok.com/@michashikes/video/7339453754955042081
- **LO-12 中国蚕桑丝织技艺**：人工=relevant → 建议=uncertain。理由：蚕丝被/地点桐乡提示相关，但需确认是否传播传统蚕桑丝织技艺/工艺过程；仅成品/卖货不算。该理由不足以判 relevant，建议 uncertain。 URL: https://www.tiktok.com/@mb999222/video/7434537422718864658
- **LI-03 黎族传统纺染织绣技艺**：人工=relevant → 建议=irrelevant。理由：黎族餐厅/少数民族料理/人物不是“黎族传统纺染织绣技艺”本体；技艺 vs 泛文化/产品边界，按新口径不算。 URL: https://www.tiktok.com/@yueh1108/video/7583368998163893525
- **LI-11 粤剧**：人工=relevant → 建议=uncertain。理由：Yue opera/yueju 不能按标签读音猜；该条理由仅写“演绎红楼梦”，未能区分粤剧/越剧。按新口径分不清标 uncertain。 URL: https://www.tiktok.com/@yunq1410/video/7531492919740599570
- **LI-17 南音**：人工=relevant → 建议=irrelevant。理由：Nanguan/南关兰州民俗节小吃不是南音 nanguan music；按项目边界不跨项/同名地名不算。 URL: https://www.tiktok.com/@hellohuda_/video/7566665746273651989
