## 1. 先把"衡量什么"钉死（防止跑偏）

这是全篇诚实性的根，必须在开篇一句话 + 结尾一句话里点到，但**不做成单独"方法与边界"章节**。

- **我们测的是「海外能见度 / 被看见的程度」，不是「影响力」。** 影响力（被打动、记住、
  态度转变、长期接受）测不了；曝光是影响力的必要前提，是有边界但站得住的代理指标。
- **脊梁指标 = 真实触达**（人工锚定的 feed 播放量 = 确属中国非遗的内容被刷到的量）。
  集体塌缩、尾部档案馆，都按"真实触达"说话。
- **稀释度不是标尺，是透镜。** 它不给非遗影响力排名，也**不预设任何词本该属于中国**
  （papercut / calligraphy 本就是各国共有的全球品类，中国只是成员之一）。它只干两件诚实的事：
  1. **校正**——把撞词灌水的虚高挤掉，得到真实触达（下界）。
  2. **能见度 = 绝对真实播放（标尺）。下面两个因子只是解释它高低的透镜，本身不替代它**（终态，2026-06-27）：
     - **有没有锚点 → 解释你的"份额"（独占赛道还是被挤），即四象限的哪一行**。份额高的几乎都靠
       **地域/民族/节庆的专属复合名**把自己从通用词里拎出来（`Cantonese opera` 粤剧 / `Tibetan medicine` 藏医 / `celadon`→`龙泉` 52%→73%）；
       **罗马音救不了份额**（`fuchuan`/`shufa`/`taichi` 份额都低），**纯西化通用译词被挤**（`papercut`/`calligraphy`）。名字是**结构属性，不是营销杠杆**。
       - ⚠️ **但份额高 ≠ 被看见的多**：太极份额仅 20%、绝对 **4100 万**，反而 ≥ 份额 76% 的藏医（绝对 **3300 万**）。
         所以锚点解释的是"挤不挤"，**不是"被不被看见"**——后者只看绝对量。
     - **内容竞争力 → 解释在大赛道、低份额下还能不能捞到大"绝对量"**（书法 8500 万 vs 剪纸 1,414）。
     - ⚠️ 不用 `junk`/`watertight` 当任何名字例子——它们是**我们的查询错误词**（已剔除认账），不是合法译词案例。
- **三条铁律（防再次跑偏，2026-06-27 终态）**：
  1. **绝对真实播放=标尺，份额=透镜，绝不用份额给隐形排名。** 否则会把被 8500 万次真实播放看见的书法
     （份额仅 12%）误判成"隐形"。份额只回答"在不在大赛道里挤"，绝对量才回答"到底多少人看见"。
  2. **2D 四象限**（绝对×份额）：①独占且被看见(15项,粤剧/京剧/太极)②挤着但被看见(4项,书法2.06亿/呼麦/宣纸/珠算)
     ③独占但小众(18项,藏戏/藏医/福船,垄断但盘子小)④真隐形(7项,剪纸/云锦/格萨尔,既不独占也没量)。每格病因对策都不同。
  3. **低占比 ≠ 身份失败，且竞争力一律用播放份额不用撞词率（条数）**：条数上 papercut 99%、calligraphy 93% 看着一样，
    但中国播放份额一个 **0%**(剪纸,绝对1,414)、一个 **12%**(书法,绝对8500万)，天差地别——差别在**内容竞争力**。
- **已知盲区**（化进结尾，不堆砌）：关键词只能发现"带文本线索"的内容；纯画面、零文案的
  中国非遗我们看不到——所以真实触达是**下界**。

## 2. 视觉底座（已定）

- 暖调深色 + 宋体大标题 + **手绘 SVG 文化插画/图标**（福船 / 剪纸 / 太极…）+ 宣纸/水墨纹理。
- 图表全部「**一图一结论 + 大白话标注**」：箭头、圈注、口语解释。
- 退黑话：「触达区间」→「真正被认出的有多少」；「撞词率」→「搜到的里有多少根本不是它」。
- 卡片化、互动、章节有节奏（深色 ↔ 朱砂色块交替）。
- **纯 vanilla JS + 内联 SVG，自包含、可离线、无 JS 也能读**。

## 4. 两个招牌交互（机制）

- **识别实验**：每轮一个**全球共有品类词** → 4–8 张真实卡（按真实播放排序）→ 读者找"中国版在哪/排第几" →
  揭晓**绝对真实播放 + 份额 + 最高排名**（已结合 378 核查）：papercut **1,414**/0%/141位 ·真隐形；
  calligraphy **8500万**/12%/第3 ·挤着但被看见；abacus **9900万**/26%/第4（核查回补了机器漏检的中国珠算）。
  - 撞词率（条数：papercut 99% / calligraphy 93%）只用来说"你刷到的大多不是它"，**能见度一律先看绝对量再看份额**。
  - 已验证真实素材："Papercut – Linkin Park"、"your body repairs a paper cut"、英文练字、沙画书法、
    日本書道/そろばん等，均来自 row_labels 真实 desc。
- **身份证档案馆**：44 项全量数据装进卡，**按四象限/绝对真实播放排**；CSS 翻面看「名字 + 真实触达 + 象限诊断 + 一句话故事」。让评委亲手翻遍数据。
