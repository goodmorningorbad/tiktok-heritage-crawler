01 研究问题与口径

研究单元是 44 个 UNESCO 中国非遗项目，而不是单条视频或单个搜索词。目标是建立项目级传播画像，并判断其在美区平台上的可见性与触达状态。

象限解释研究含义
存量高 × 触达高内容多，且有人看真·规模化出海候选
存量高 × 触达低内容多，但没人看自产自销 / 虚假繁荣重点观察位
存量低 × 触达高内容少，但有爆点小而精破圈
存量低 × 触达低内容少，也没人看近乎隐形
口径红线存量回答“有多少内容被生产”，触达回答“内容有没有被看到”。TikTok hashtag videoCount 与 YouTube 采集状态不是同一单位，不能数值硬比；可比的是 relevance-aware 触达口径。

05 已知局限

  • TikTok search 有隐藏翻页上限，高存量词不能用 search collected_count 横向排名。
  • TikTok hashtag videoCount 有撞词噪声，必须保留 clean / noisy / unavailable 状态。
  • YouTube 3 页 baseline 是浅采下限，56 个 sample_limit_reached 不代表平台天花板。
  • YouTube Y3 P0 深采目前是 partial，8 个 failed 词等待明日补跑。
  • 受众只能用评论区语言作有偏代理，不做精确族裔比例判断。
  • 当前 regionCode / 采集出口以 US 为主，结论限于美区语境。
