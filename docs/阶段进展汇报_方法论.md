# UNESCO 中国非遗在 TikTok / YouTube 美区的传播研究
## 阶段进展与方法论汇报（更新版）

> 汇报定位：**研究进展 + 方法论**，非最终成果展示。  
> 更新时间：2026-06-16 晚；因汇报推迟到明天，本版补入今日完成的 TikTok 相关性清洗、存量×触达草表、YouTube 3 页扩采、Y2 深采候选与 Y3 P0 部分深采结果。  
> 核心原则：**内容存量 ≠ 传播触达；平台间存量不可数值硬比；噪声只标不删；技术失败不等于低可见。**

---

## 一、研究问题与当前口径

### 1.1 研究问题

以 UNESCO 认定的 44 个中国非物质文化遗产项目为单位，判断它们在 TikTok / YouTube 美区的可见性与传播触达状况，重点区分：

- 哪些项目是 **存量高 + 触达高**：真·规模化出海候选；
- 哪些项目是 **存量高 + 触达低**：大量产出但无人问津，可能是自产自销 / 虚假繁荣；
- 哪些项目是 **存量低 + 触达高**：小而精破圈；
- 哪些项目是 **存量低 + 触达低**：近乎隐形。

### 1.2 研究对象

- 研究单元：**非遗项目**，不是单条视频，也不是单个搜索词。
- 当前名录口径：44 个项目。
- 名录类别：RL 36、USL 7、GSP 1。
- 2025 新增 / 转列暂不纳入本轮，作为边界声明。

### 1.3 核心方法论纠偏

| 指标类型 | 代表数据 | 回答的问题 | 不能回答的问题 |
|---|---|---|---|
| 存量 | hashtag videoCount、采集触顶、视频条数 | 有多少内容被生产出来 | 是否真的被看见 |
| 触达 | playCount、like、comment、share | 内容有没有被看到 | 内容总量有多大 |
| 受众代理 | 评论区语言 | 谁可能在看 / 互动 | 精确族裔比例 |

因此，后续所有结论都按 **存量 × 触达** 的二维框架推进，而不是按单一热度或单一视频数量排序。

## 二、TikTok 已完成进展

### 2.1 baseline 数据基底

- baseline run：2026-06-12。
- 原始视频：32,579 条。
- 去重唯一视频：29,407 条。
- 搜索词：197 个。
- 技术失败：0。

早期关键发现仍保留：TikTok search 接口存在约第 6–7 页的隐藏翻页上限，热门词会被压平在约 180–200 条；因此 search collected_count 只能作为内容存量的弱信号，不能直接横向排名。

### 2.2 相关性口径 v1 已工程化

已将历史人工核查口径应用到当前 baseline，生成三档标签：

| 标签 | 数量 | 含义 |
|---|---:|---|
| likely_relevant | 10,111 | 高置信相关 |
| needs_review | 4,053 | 边界 / 待人工确认 |
| low_relevance | 18,415 | 低相关 / 噪声风险 |

关键点：**low_relevance 不删除**，而是作为噪声风险和语义稀释信号保留。

### 2.3 TikTok 触达维重算

已按项目重算四套触达：

- raw：所有视频；
- likely：只看 likely_relevant；
- inclusive：likely + needs_review；
- low：低相关噪声单独保留。

这一步解决了 raw reach 被撞词和泛词视频抬高的问题。

### 2.4 hashtag 存量轴已补齐

已采集 TikTok hashtag statsV2.videoCount，项目级状态：

- head：5 项；
- mid：5 项；
- long_tail：8 项；
- near_invisible：26 项；
- 同时保留 clean / noisy / unavailable 数据状态。

### 2.5 TikTok 存量 × 触达草表

已形成 44 项目的初版二维矩阵：

| 象限 | 项目数 | 含义 |
|---|---:|---|
| stock_high__reach_high | 7 | 真·规模化出海候选 |
| stock_high__reach_low | 3 | 高存量低触达，重点观察自产自销 / 虚假繁荣 |
| stock_low__reach_high | 10 | 小而精破圈候选 |
| stock_low__reach_low | 24 | 近乎隐形候选 |

## 六、已知局限（汇报中应主动说明）

1. TikTok search 有隐藏翻页上限，高存量词不能用 search collected_count 横向排名。
2. TikTok hashtag videoCount 有撞词噪声，必须保留 clean / noisy / unavailable 状态。
3. YouTube 3 页 baseline 是浅采下限，56 个 sample_limit_reached 不代表平台天花板。
4. YouTube Y3 P0 深采目前是 partial，8 个 failed 词等待明日补跑。
5. 受众只能用评论区语言作有偏代理，不做精确族裔比例判断。
6. 当前 regionCode / 采集出口以 US 为主，结论限于美区语境。
